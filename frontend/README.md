# 答案之书前端应用

## 技术栈
- Flutter
- Material Design

## 目录结构
```
/frontend
├── lib/
│   ├── main.dart           # 应用入口
│   ├── models/             # 数据模型
│   ├── screens/            # 页面
│   ├── widgets/            # 自定义组件
│   ├── services/           # 服务
│   └── utils/              # 工具类
├── assets/                 # 静态资源
├── test/                   # 测试文件
└── pubspec.yaml           # 依赖配置
```

## 开发规范
1. 遵循Flutter官方代码规范
2. 使用Material Design组件
3. 实现响应式设计
4. 编写单元测试

## 主要功能
1. 书本展示界面
2. 翻书动画效果
3. 答案展示页面
4. 主题切换
